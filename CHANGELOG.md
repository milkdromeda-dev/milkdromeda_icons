## 0.0.3+7

Added 301 new **badge-style** icons (3rd visual variation) across three new font sets:

- **Set 7** — badge variants of all Set 1 / Set 2 icons (action, bell, dolphin, snail, camera, calendar, …)
- **Set 8** — badge variants of all Set 3 / Set 4 icons (airplane, bicycle, crown, chess pieces, musical notes, …)
- **Set 9** — badge variants of all Set 5 / Set 6 icons (beds, antiseptic, nurse, cardiologist, …)

Each badge icon renders the original shape as a white negative-space cutout inside a solid rounded-square, giving a modern contained-icon look ideal for navigation bars, chips, and action buttons.

New icon names follow the `*3` convention (e.g. `MilkdromedaIcons.dolphin3`, `MilkdromedaIcons.snail3`, `MilkdromedaIcons.airplane3`, `MilkdromedaIcons.beds3`).

Total icon count: **903** across 9 font sets.
